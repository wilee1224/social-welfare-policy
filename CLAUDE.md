# 사회복지정책 알리미 하네스

**목표:** 국내 사회복지 정책 변경사항을 매일 수집·분석·검증하여 웹사이트에 발행

**트리거:** 복지정책 수집, 사이트 업데이트, 오늘 정책 확인 등 사회복지 관련 작업 요청 시 `welfare-orchestrator` 스킬을 사용하라. 단순 질문은 직접 응답 가능.

**프로젝트 구조:**
- `index.html` + `css/style.css` + `js/app.js` — 정적 웹사이트
- `data/policies.json` — 정책 데이터 (에이전트가 업데이트)
- `.claude/agents/` — 6개 에이전트 정의
- `.claude/skills/` — 5개 스킬 (orchestrator 포함)
- `_workspace/` — 실행 중간 산출물
